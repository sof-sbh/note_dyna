# 5. Expérience utilisateur

## Fonctions principales

| Fonction | Description |
|---|---|
| **RUM** | Observe les interactions de vrais utilisateurs. |
| **Synthetic Monitoring** | Exécute des tests automatisés simulant des utilisateurs ou appels HTTP. |
| **Session Replay** | Reconstitue visuellement une session utilisateur selon la configuration et les règles de confidentialité. |

## Types d’actions utilisateur

| Action | Description |
|---|---|
| **Load actions** | Mesurent le chargement de pages, notamment le temps jusqu’au premier octet et les indicateurs visuels. |
| **XHR actions** | Mesurent les requêtes asynchrones exécutées par le navigateur. |
| **Custom actions** | Permettent de suivre des interactions ou événements définis spécifiquement. |

## Différence RUM vs Synthetic

- **RUM** : comportement réel d’utilisateurs réels.
- **Synthetic** : scénarios automatisés exécutés de façon planifiée.
