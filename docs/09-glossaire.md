# 9. Glossaire

| Terme | Définition |
|---|---|
| **APM** | Application Performance Monitoring. |
| **RUM** | Real User Monitoring. |
| **SLI** | Service Level Indicator. |
| **SLO** | Service Level Objective. |
| **RPO** | Recovery Point Objective. |
| **RTO** | Recovery Time Objective. |
| **MTTD** | Mean Time To Detect. |
| **MTTR** | Mean Time To Repair ou Recover. |
| **Trace** | Ensemble de spans décrivant une transaction distribuée. |
| **Span** | Opération élémentaire dans une trace. |
| **PurePath** | Représentation détaillée d’une transaction suivie par Dynatrace. |
| **Flapping** | Ouverture et fermeture répétées d’une alerte instable. |
